Introdução ao HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar páginas da web. Ela consiste em uma série de elementos que são usados para estruturar o conteúdo de uma página. Neste texto, vamos explorar alguns elementos-chave do HTML, passando por eles em uma ordem didática.

< html > - Elemento HTML Raiz

O elemento < body > contém todo o conteúdo visível da página, como texto, imagens e outros elementos HTML. É aqui que a maior parte do conteúdo da página é colocada.

< h1 >, < h2 >, até < h6 > - Títulos e Subtítulos

Esses elementos são usados para criar títulos e subtítulos. < h1 > é o título mais importante, enquanto < h6 > é o menos importante. Eles ajudam a estruturar e hierarquizar o conteúdo da página.

< strong > - Texto em Destaque

O elemento < strong > é usado para dar ênfase ao texto, geralmente tornando-o em negrito. É útil para destacar palavras ou frases importantes.

< mark > - Texto Marcado

O elemento < mark > é usado para destacar parte do texto, geralmente com um fundo amarelo. É útil para indicar algo importante em um bloco de texto.

< blockquote > - Citações

O elemento < blockquote > é usado para citar o texto de outra fonte. Ele geralmente é exibido com recuo e é útil para indicar que o texto é uma citação.

< a href="" > - Links

O elemento é usado para criar links para outras páginas ou recursos na web. O atributo href especifica o destino do link.

Estes são apenas alguns dos elementos HTML básicos que você pode usar para criar páginas web. À medida que você avança em sua jornada de aprendizado, você descobrirá muitos outros elementos e recursos que o HTML tem a oferecer. Lembre-se de que a prática é fundamental para aprofundar seu conhecimento em HTML.
